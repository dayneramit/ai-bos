# AI-BOS Project Context

## Project Information

Project Name:
AI-BOS (Artificial Intelligence Business Operating System)

Repository:
dayneramit/ai-bos

Purpose:
สร้างระบบ Business Operating System สำหรับบริหารธุรกิจแบบครบวงจร
รวมระบบงานบริการ งานขนส่ง เอกสาร การเงิน และ Automation ไว้ในแพลตฟอร์มเดียว

---

# Project Vision

เป้าหมายของ AI-BOS:

- ลดงาน Manual
- ลดความผิดพลาดในการทำงาน
- เก็บข้อมูลธุรกิจอย่างเป็นระบบ
- รองรับการขยายทีม
- ใช้งานจริงระดับ Production
- รองรับ Mobile First Workflow
