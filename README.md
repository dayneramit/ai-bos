# AI-BOS

Artificial Intelligence Business Operating System

## Overview

AI-BOS is a production-ready business operating platform designed to manage service operations, logistics, documents, finance, and automation.

## Applications

### Day Neramit

Service business management system:

- Customer booking
- Technician management
- Quotation
- Invoice PDF
- Job tracking
- Document automation


### NC Logistics

Logistics management system:

- Admin dashboard
- Driver application
- Booking system
- Check-in / Check-out
- GPS tracking
- Job history
- Income calculation


## Technology Stack

- Next.js
- TypeScript
- Tailwind CSS
- Firebase
- Google Apps Script
- Vercel


## Development Principle

This project follows:

- Production Ready
- Secure Architecture
- Scalable Design
- Mobile First
- Clean Code
